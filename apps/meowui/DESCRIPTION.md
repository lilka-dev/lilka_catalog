# Meowui

Meowui — це фреймворк, призначений для розробки прошивок для мікроконтролерів ESP32 та ESP32-S3.

Framework designed for developing firmware for ESP32 and ESP32-S3 microcontrollers.

## Особливості

Прошивка збирається плагіном [Pioarduino](https://github.com/pioarduino) та не може бути зібрана з допомогою Platformio. Прошивку налаштовано для плати [Лілка v2.2+](https://lilka.dev/). Щоб зібрати прошивку для власної плати, видаліть/відредагуйте приклади контекстів та налаштуйте параметри в заголовкових файлах в каталозі `include`.

> **Note**  
> Приклади контекстів в цьому репозиторії наразі не актуалізовані і можуть містити застарілий код. Їх наявність тут необхідна для демонстрації можливостей Meowui на популярній платі ЛілкаV2. Оновлені приклади, які будуть підтримуватися в майбутньому, можна знайти в репозиторії [Ідеальної консолі](https://github.com/Kolodieiev/perfect_console).
