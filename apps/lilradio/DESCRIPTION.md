# Lilradio

FM radio receiver module for [Лілка](https://docs.lilka.dev/)

### Parts

- RDA5807M FM-receiver module
- PAM8302 audio amplifier module
- Speaker 1635
- SMA connector with wire
- SMA Antenna
- 40 Pins Male 2.54mm Right angle connector
- 2x M3x5, 4x M2x5 screws

## Case

STL files of the case for 3D printing are located in the stl folder

## Controls

- Right/Left buttons - change frequency up/down
- Up/Down buttons - change volume up/down
- Select button - mute/unmute volume
- A/D buttons - radio seek up/down

The radio module stores the last used frequency and volume setting
