# Хрестики-нулики для Лілки

Це реалізація класичної гри "Хрестики-нулики" для платформи Лілка. Гравець змагається проти штучного інтелекту, який має базову логіку з елементами випадковості. Гра включає анімацію виграшу, пульсуюче підсвічування вибраної клітинки та відображення рахунку перемог для кожного гравця.

## Особливості

- **Гра проти AI**: Штучний інтелект має базову логіку з елементами випадковості
- **Анімація виграшу**: Перекреслення виграшної лінії
- **Підсвічування вибраної клітинки**: Пульсуюча зелена рамка
- **Рахунок**: Відображення кількості перемог для кожного гравця
- **Підтвердження виходу**: Можливість вийти з гри після підтвердження
- **Відтворення звуку при програші**: Сумна мелодія відтворюється, якщо виграв гравець "O"
- **Відтворення звуку при виграші**: Радісна мелодія відтворюється, якщо виграв гравець "X"

## Управління

- **Стрілки**: Переміщення між клітинками
- **A**: Зробити хід / підтвердити вихід
- **B**: Скасувати вихід / почати нову гру після завершення
- **D**: Ініціювати вихід із гри

## Як запустити

Щоб запустити гру, потрібно завантажити файл `tic_tac_toe_lilka.lua` на Лілку. Це можна зробити за допомогою FTP-сервера та клієнта, наприклад, Filezilla. Після завантаження файл можна запустити через інтерфейс меню Лілки.
