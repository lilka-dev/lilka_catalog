# Incident LightMeter for Lilka

Базова версія експонометра для вимірювання падаючого світла.
Використовується датчик BH1750 та напівсфера від ІЧ-Датчика HC-SR501.

Підключення: 
VCC=3V3, GND=GND, pin12=scl, pin13=sda

C: Змінює iso, A: Змінює діафрагму, Select: Вихід
